# Routing Engine

**Owner:** Data Science Team | **Status:** Active | **Language:** Python

## Purpose
Core ML-based dynamic routing engine taking into account vehicle constraints, real-time traffic updates, and SoC (State of Charge).

## Key Features
- Dynamic re-routing based on vehicle failure or congestion points.
- ML models predicting battery drain across various terrains.

## Quick Start
```bash
git clone https://github.com/opeva/routing-engine
cd routing-engine
pip install -r requirements.txt
```
