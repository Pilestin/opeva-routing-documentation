# Merkezî Repolar & Kod Tabanları

OPEVA projesi ve ilgili araç rotalama (VRP) araştırmaları kapsamında geliştirilen tüm açık kaynak repolar aşağıda kategorize edilmiştir. Bu repolar, projelerin kaynak kodlarını, simülasyon ortamlarını ve kullanıcı arayüzlerini barındırır.

## 🧠 Optimizasyon ve Yapay Zeka (Routing & AI)

| Repository | Açıklama | Teknoloji |
|------------|----------|-----------|
| **[vehicle-routing-problem...](https://github.com/Pilestin/vehicle-routing-problem-from-the-perspective-of-customer-satisfaction)** | Müşteri memnuniyetini ön plana alan, zaman pencereli Elektrikli Araç Rotalama (ALNS) algoritması kaynak kodları. | Python |
| **[DeepVRP](https://github.com/Pilestin/DeepVRP)** | Araç rotalama problemlerini Derin Öğrenme (Deep Learning) teknikleriyle çözen metotlar deposu. | Python |
| **[GnnVrp](https://github.com/Pilestin/GnnVrp)** | Graf Nöral Ağları (GNN) ile Araç Rotalama Problemine giriş ve denemeler. | Python |
| **[vrp-master](https://github.com/Pilestin/vrp-master)** | Genel VRP problem çözümü kod havuzu. | Python |
