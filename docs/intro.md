# Giriş

**Elektrikli Araç Rotalama Problemleri (EVRP)** ve **Routing Markup Language (RoutingML)** çalışmalarına hoş geldiniz.

## Bu Dökümantasyon Hakkında

Bu dökümantasyon, elektrikli araç rotalama problemleri ve çözüm yöntemleri üzerine yapılan araştırma ve geliştirme çalışmalarını kapsamaktadır.

### Ana Konular

- **Elektrikli Araç Rotalama Problemleri (EVRP)**
- **Routing Markup Language (RoutingML)**  
- **ALNS Algoritması Uygulamaları**
- **Filo Yönetim Sistemleri**
- **Client-Server Mimarisi İmplementasyonları**

### Çalışma Alanları

🔋 **Elektrikli Araç Optimizasyonu**
- Batarya kapasitesi constraints
- Şarj istasyonu planlaması
- Enerji tüketim modelleme

📊 **Routing Markup Language**
- Standart girdi/çıktı formatları
- XML/JSON şema tanımları
- API standardizasyonu

🔧 **Algoritma Geliştirme**
- ALNS (Adaptive Large Neighborhood Search)
- Meta-heuristik yaklaşımlar
- Performans optimizasyonu

🏢 **Filo Yönetimi**
- Gerçek zamanlı takip
- Route optimizasyonu
- Maliyet analizi
